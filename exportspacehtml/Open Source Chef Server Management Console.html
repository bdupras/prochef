Chef : Open Source Chef Server Management Console
This page last changed on Nov 27, 2011 by tom@opscode.com.



The Management Console is Chef Server's web interface.

Nodes, roles, cookbooks, data bags, and API clients can be managed through the Management Console. Search can also be done on the console.

Setting up the Management Console

In order to set up and start the management console, use the chef::bootstrap_server recipe with webui_enabled. The management console will be running on port 4040.

Using the Management Console

In order to start using the Management Console, you need to first create a user or change the default password on the "admin" user.
